# 🌌 Hextech Portfolio | Dhoby Bumacod

A premium, interactive professional portfolio inspired by the **"Arcane Architect"** aesthetic. Designed with high-contrast Hextech filaments (Cyan & Gold), glassmorphism, and dynamic luminous occlusion.

## 🛠️ Performance & Tech Stack
- **Core:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS (Glassmorphism, Advanced Motion Keyframes)
- **Architecture:** Data-driven constants for Expertise, Timeline, and Projects
- **Dev Tools:** ESLint, HMR, Oxc

## ✨ Core Features

### 🌪️ The Arcane Portal (Hero Section)
A high-energy, interactive portal built with multiple concentric SVG/CSS rings. 
- **Erratic Movement:** On hover, the subject is illuminated by five layers of asynchronous, non-linear rotating energy fields.
- **Dynamic Introduction:** Professional summary blending software engineering, data streaming, and autonomous AI system design.

### 📜 Professional History
A serialized, reactive timeline highlighting key roles at **Arch Global Services** and **Globe Group**.
- **Tactile Hover:** Timeline gems and structural borders react to proximity with intense focused glows and organic translations.

### 🧪 The Arsenal (Skills & Expertise)
A recessed-container grid categorizing proficiency across Development, Infrastructure, Data, and AI Engineering.
- **Micro-Interactions:** Hover states simulate physical depth through inner-shadow manipulation and ambient border lighting.

### 🛸 Independent Ventures
Technical project showcases (e.g., Kafka Order Processing, AI Personal Assistant) extracted from raw git READMEs.
- **Levitating Glass Cards:** Elevated hover states with atmospheric brand-colored shadows (Cyan/Gold) to separate specific project domains.

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Dev Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```
